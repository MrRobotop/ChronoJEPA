# Paper sources

LaTeX sources for the ChronoJEPA working paper.

- `paper.tex` is self-contained: it uses the standard `article` class and common packages, so it
  compiles with any TeX distribution without venue style files.
- `references.bib` holds the bibliography. The LeJEPA entry has a note: verify its author list and
  exact title against arXiv:2511.08544 before submission.
- Figures are read from `../figures/` (generated by the repository scripts; see the main README and
  RESULTS.md).

## Build

```bash
pdflatex paper
bibtex paper
pdflatex paper
pdflatex paper
```

(or `latexmk -pdf paper`).

## Retargeting to a venue

Replace the document class and geometry with the venue style, leaving the body unchanged. For
example, with `neurips_2024.sty` alongside this file:

```latex
\documentclass{article}
\usepackage{neurips_2024}
```

or for ICLR with `iclr2025_conference.sty`:

```latex
\documentclass{article}
\usepackage{iclr2025_conference,times}
```

The prose, tables, figures, and bibliography are venue-agnostic.
